# Coinstr SDK - Python Package

The Python language bindings for [coinstr-sdk](https://github.com/coinstr/coinstr).

## Getting started

```shell
pip install coinstr-sdk
```

## State

**This library is in an ALPHA state**, things that are implemented generally work but the API will change in breaking ways.
